# Happy Habits — Project Rules

## Запрещённая лексика (СТРОГО)

На сайте **НИКОГДА** не должны появляться следующие слова и формулировки:

### Медицинские термины (Ксения — клинический психолог, НЕ врач)
- **психотерапевт** / **психотерапия** — использовать: «психолог», «КПТ-специалист», «психологическая помощь»
- **диагноз** — использовать: «запрос», «ситуация», «состояние»
- **лечение** (тревоги / депрессии / невроза и т.д.) — использовать: «работа с», «помощь при»
- **депрессия** — использовать: «подавленное состояние», «апатия», «подавленность»
- **невроз** — использовать: «тревожное состояние», «повышенная тревожность»
- **расстройство** / **расстройства** — использовать: «тревожность», «состояние»
- **симптомы** — использовать: «признаки», «проявления», «сигналы»
- **панические атаки** (как диагноз) — использовать: «приступы тревоги», «внезапная тревога»

### Гарантии результата (нарушение рекламного законодательства)
- **избавлю** / **избавление** / **избавиться** — использовать: «снижение», «работа с», «освобождение от гнёта»
- **вылечу** / **вылечить** / **лечение** — использовать: «помогу справиться», «работа над»
- **гарантирую результат** — использовать: «по данным исследований, КПТ помогает 7-8 из 10 клиентов»

### Правильная должность
- Ксения = **клинический психолог, КПТ-специалист**
- В JSON-LD jobTitle: `"Клинический психолог, КПТ-специалист"`
- Никогда: «психотерапевт», «врач», «доктор»

## Дисклеймер (футер)
Формулировка: «Я не работаю с клиническими состояниями, требующими медикаментозного лечения» — без слов «депрессия», «психоз» и подобных.
