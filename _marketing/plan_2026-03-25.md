**Почему не 70₽:** при 70₽ рискуем потерять часть показов в конкурентных аукционах. Лучше снижать постепенно: 120₽ → 90₽, через 2 недели оценить.

### Фаза 4 (26-28 марта): Консолидация лендинга

*Выполняется сразу — не влияет на рекламную кампанию.*

#### 4.1 Убрать вариант B, оставить только A

**Что:** отключить A/B роутинг, направлять весь трафик на вариант A.

**Почему:**
- При 3 конверсиях за 7 недель A/B тест статистически бессмыслен (нужно 100+ конверсий на вариант)
- Рандом заголовков (ab-test.js: 4×4×4 = 64 комбинации) поверх рандома вариантов — хаос
- Двойной редирект (root → /v/a|b/) замедляет загрузку на мобильных
- Каждое изменение нужно делать дважды (A = партиалы, B = инлайн)

**Почему A, а не B:**
- A использует партиалы → легко менять каждую секцию
- B — инлайн-контент в одном шаблоне → сложнее поддерживать
- ab-test.js данные накоплены только для A

**Из B берём лучшее:**
- Эмоциональные элементы текста можно перенести в A если понадобится
- Тёплая тема (warm) остаётся доступной через CSS

**Технически:**
- `config.toml`: убрать вариант B из списка, или отключить роутинг
- `layouts/index.html`: редирект сразу на /v/a/ без рандома
- Файлы B можно не удалять (оставить для истории), просто не показывать

#### 4.2 Отключить рандом заголовков (ab-test.js)

**Что:** зафиксировать один заголовок + подзаголовок + CTA вместо рандома из 64 комбинаций.

**Лучшая комбинация (по данным параметров визитов Метрики):**
- Заголовок: любой из 4 (все примерно равны: 3-5 конверсий каждый)
- Подзаголовок: **«Когнитивно-поведенческая терапия — от первой встречи к ощутимым переменам.»** (8 конверсий — лидер)
- CTA: все 4 равны (по 4 конверсии). Оставляем **«Записаться на вводную встречу»** — самый прямой и понятный.

**Заголовок фиксируем:** «Тревога мешает жить? Я помогу это изменить.» — совпадает с лучшим объявлением в Директе (CVR 17%), обеспечивает согласованность объявление → лендинг.

**Технически:** отключить ab-test.js, прописать фиксированные тексты в шаблоне hero.

#### 4.3 Добавить мини-пакет на лендинг

Было:
```
Разовая сессия:    10 000₽ → 5 000₽  (60 мин)
Пакет 10 сессий:  100 000₽ → 45 000₽  (4 500₽/сессия)
```

Стало:
```
Вводная встреча:   бесплатно (30 мин)           ← порог входа
Разовая сессия:    10 000₽ → 5 000₽ (1ч 20мин)  ← для осторожных
Пакет 4 сессии:   20 000₽ → 16 000₽             ← 4 000₽/сессия, экономия 20%
Пакет 10 сессий:  50 000₽ → 38 000₽             ← 3 800₽/сессия, экономия 24%
```

**Что изменилось:**
1. Добавлен мини-пакет 4 сессии — снижает порог входа (16 000₽ vs 38 000₽)
2. Пакет 10 пересчитан: 38 000₽ вместо 45 000₽ — теперь лестница скидок логична (5 000 → 4 000 → 3 800)
3. Якорь пакета 10 изменён с 100 000₽ на 50 000₽ (10 × 5 000₽) — реалистичнее
4. Время сессии увеличено с 60 мин до **1 часа 20 мин** — конкурентное преимущество
5. ~~Пакет 10 действует 4 месяца вместо 3~~ → Убраны сроки действия пакетов («Действует N месяцев») — убирает ощущение давления и упрощает карточки

#### 4.4 НЕ менять цену разовой сессии

Снижение с 5 000₽ до 3 900₽ **не рекомендуется:**
- Разница 1 100₽ не меняет сегмент (обе «дорого» для чувствительных к цене)
- Теряем премиум-позиционирование
- Попадаем в средний сегмент, где конкуренция с агрегаторами (Ясно, Alter — 2 000-3 500₽, 100+ психологов на выбор)
- Снижение маржи: для безубыточности нужно 11 сессий вместо 8
